# Default-SHR
Default Shared Health Record Service

## Setup
1. Edit the following files:
    * ```nginx/default-shr-nginx```
    * ```nginx/default-shr-nginx.conf```
    * ```upstart/default-shr-upstart.conf```
    * ```config.py```
    * ```install.sh```
2. Run ```install.sh```
    ```
    ./install.sh
    ```
